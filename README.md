# app-z-to-dht

Decrypts, extracts and converts data from Z*lo to [Discord History Tracker](https://dht.chylex.com/) format for viewing.

Requires a decryption key, obtained using `get_dkey.py` from [zalo-decrypt-backup](https://github.com/beer-psi/zalo-decrypt-backup).
